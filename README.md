# Secure File Encryptor 🛡️

A Python-based security tool that uses **AES-256 (Fernet)** symmetric encryption to protect sensitive data. Created as part of my Cybersecurity portfolio at the University of Fort Hare.

## Features
* **Key Generation:** Creates a unique 256-bit encryption key.
* **Symmetric Encryption:** Uses the same key to lock and unlock data.
* **Safe Handling:** Converts strings to bytes for cryptographic processing.

## How to Run
1. Install dependencies: `pip install cryptography`
2. Run `encryptor.py` to generate a key and encrypt a message.
